# Bases de JavaScript

Este directorio agrupa ejercicios básicos de JavaScript para aprender los fundamentos del lenguaje, desde objetos y clases hasta funciones, promesas y manejo de errores.

## Secciones incluidas

- `Objetos/` - Ejemplos y prácticas con objetos en JavaScript.
- `Clases/` - Ejercicios con clases, herencia, getters, setters y miembros estáticos.
- `FuncionesCallBack/` - Uso de funciones callback, asincronía y temporizadores.
- `FuncionesFlecha/` - Ejemplos de funciones tradicionales, anónimas y flecha.
- `ModoEstricto/` - Ejemplos del modo estricto (`"use strict"`) y su efecto en el código.
- `Errores/` - Manejo de errores con `try`, `catch`, `finally` y creación de errores personalizados.
- `Promesas/` - Uso de promesas, `async`, `await` y control de operaciones asíncronas.

## Cómo usar esta carpeta

Cada subcarpeta contiene un `README.MD` propio con la explicación del ejercicio y los archivos correspondientes. Navega a la sección que te interese y ejecuta los ejemplos con Node.js.

Por ejemplo, para ejecutar el ejercicio de promesas:

```powershell
cd Promesas
node promesa.js
```

## Recomendación de estudio

1. Comienza por `Objetos/` para entender la base de los datos en JavaScript.
2. Sigue con `Clases/` para aprender programación orientada a objetos.
3. Continúa con `FuncionesFlecha/` y `FuncionesCallBack/` para dominar funciones y callbacks.
4. Aprende seguridad y estilo con `ModoEstricto/`.
5. Revisa `Errores/` para manejar fallos de forma segura.
6. Finaliza con `Promesas/` para trabajar con código asíncrono moderno.

## Ejercicios de práctica

- Abrir cada subcarpeta y leer su `README.MD`.
- Ejecutar los ejemplos en consola y modificar los valores.
- Probar variaciones de los ejercicios para reforzar los conceptos.
